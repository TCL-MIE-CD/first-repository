# MTT

基于FIS3的前端集成解决方案。


## Get Start

安装MTT：
```
npm install mtt -gd
```

初始化J2EE项目（默认）：
```
mtt init
mtt server start
mtt release
```
J2EE项目的demo
[http://120.25.62.254:9999/](http://120.25.62.254:9999/ "mtt-jello demo")

初始化React SPA项目：
```
mtt init webpack-react-spa
npm run mock
npm start
```
